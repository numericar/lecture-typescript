Typescript introduction
    เป็น super set ของ Javascript โดนได้เพิ่มฟังก์ชันใหม่ ๆ และปรับปรุงการทำงานจาก Javascript เดิมให้ดีขึ้น
    โดยจะทำการแปรง Typescript เป็น Javascript ก่อนนำไปใช้งาน

    โดยจะช่วยตรวจสอบข้อผิดพลาดบนชุดคำสั่งของเราด้วย เช่น การตรวจสอบประเภทของข้อมูล

    โดยสิ่งที่เพื่มใน typescript มีดังนี้
        1. ชนิดของข้อมูล
        2. สามารถใช้งานระบบใหม่ ๆ ของ javascript ได้ โดยจะสามารถใช้งานได้ทั้งในระบบรุ่นใหม่และรุ่นเก่า
        3. มีบางคุณสมบัติที่มีเฉพาะ typescript เช่น interface, generic
        4. สามารถ configuration ได้อย่างอิสระ
