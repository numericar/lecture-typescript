Data Type
    1. number   จัดเก็บเลขจำนวนเต็มและทศนิยม (int, float, double)
    2. string   จัดเก็บกลุ่มของตัวอักษร
    3. boolean  จัดเก็บค่าจริงเท็จ
    4. object   จัดเก็บข้อมูลในรูปแบบของ key:value
    5. array    จัดเก็บข้อมูลที่หลายค่า
    6. tuple    จัดเก็บข้อมูลที่หลายค่า แต่จะมีการระบุจำนวนของข้อมูลข้างในไว้ชัดเจน และระบุด้วยว่าข้อมูลแต่ละตำแหน่งมีชนิดเป็นอะไร

ข้อแตกต่างระหว่าง js และ ts
    - javascript ใช้ dynamic type 
    - typescript ใช้ static type
