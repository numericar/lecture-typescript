Type Inference
    เป็นการให้ compiler ของ typescript คอยระบุเองว่า variable/function/object ควรเป็นชนิดข้อมูลอะไร
    เช่น 
        const x = 10; compiler จะระบุให้เป็น number
        const x = "Hello World"; compiler จะระบุให้เป็น string 

Type Annotation
    เป็นการให้ developer เป็นผู้ระบุว่า variable/function/object นั้นจะต้องเป็น type อะไร
    เช่น
        const x: number = 10;
        const x: string = "Hello World";
